## 해주라
숙박 예약 서비스

## 사용법
1. 기본 모듈 패키지 설치
```
npm install
```
2. 프론트 서버 실행
```
npm run dev
```

# 구현기능
## 유저 페이지
### 숙소 검색
- [x] 숙소 검색
- [x] 숙소 검색 결과 표시
- [x] 숙소 상세 페이지 표시
### 숙소 예약
- [x] 숙소 예약 정보 미리보기 
- [x] 숙소 예약 확인 및 결제( + 상세 요금 표시)

- [ ] 숙소 리뷰 작성

## 관리자 페이지
### 관리자 홈
- [ ] 이번 달 예상 총 방문자수 표시
